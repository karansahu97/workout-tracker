Design Identity

Role: Senior UI/UX Designer & Frontend Developer

Objective: High-fidelity, mobile-first workout tracking application

Vibe: Premium, "pro-athlete" grade

Aesthetic: Light-mode with crisp contrast and subtle neon accents

Visual & Color System

Primary Theme:

Light Mode

Color Palette:

Backgrounds: Pure white (#FFFFFF) or soft light grey (#F5F5F5)

Surface/Cards: Clean neutral grey (#FFFFFF) with subtle elevation

Borders: Subtle 1px borders in black/5

Primary Accent: Lime Green (#A3E635) for CTAs and active states

Data Accents: Neon Purple, Electric Blue, Soft Orange, and Crimson

Typography:

Headers: Sans-serif, bold, uppercase with wide tracking

Data: Monospaced or high-legibility fonts for numerical values

Core Components

Sticky Header

Center-aligned title in uppercase tracking

Subtle glass / soft blur effect or clean solid background

Streak Heatmap

GitHub-style grid of small rounded squares (3x3px or 4x4px)

Month labels (horizontal) and Day labels (vertical)

Color-coded intensity based on accent palette

Workout History Cards

Header with Date/Time and Clock icon (duration)

Three-column grid: Total KCAL, Moved KG, Exercises

Pill-shaped muscle group tags (e.g., "BACK" in yellow)

Chevron-right indicator for navigation

Training Plans View

Cards with soft horizontal gradients (lime-to-transparent)

Low-contrast background dumbbell icons

Clear metadata for Duration and Intensity

Navigation & HUD (UX Critical)

Primary CTA: Large, full-width pill-shaped "Start Workout" button in Lime Green

Floating Nav: Secondary pill-shaped container with icons for Workouts, Plans, and Profile

Technical Implementation

Framework: React

Styling: Tailwind CSS

Iconography: Lucide-React

Responsiveness: Max-width container (max-w-xl) for mobile-app feel on all devices

Motion: Smooth tab transitions and subtle scale-in card animations
